# 实验室管理系统 (Laboratory Management System)

一个基于 Vue 3 + Vite 构建的现代化实验室管理系统，提供完整的实验室资源管理、申请流程、设备维修等功能。

## 🚀 技术栈

### 前端框架

- **Vue 3** - 渐进式 JavaScript 框架，使用 Composition API
- **Vite** - 下一代前端构建工具，快速热重载
- **Vue Router** - 官方路由管理器
- **Pinia** - Vue 3 状态管理库

### UI 组件库

- **Ant Design Vue** - 企业级 UI 设计语言和 Vue 组件库
- **SCSS** - CSS 预处理器，支持嵌套和变量

### 开发工具

- **ESLint** - 代码质量检查工具
- **Prettier** - 代码格式化工具
- **Vue DevTools** - Vue 开发调试工具

## 🎯 核心功能

### 1. 用户权限管理

- **角色区分**：教师账号 vs 学生账号
- **权限控制**：基于角色的功能访问控制
- **用户认证**：登录验证和会话管理

### 2. 实验室管理

- **实验室信息管理**：添加、编辑、查看实验室详情
- **设备管理**：设备信息维护、状态跟踪
- **负责人管理**：教师负责的实验室权限控制

### 3. 申请流程系统

- **在线申请**：学生可在线提交实验室使用申请
- **申请审核**：教师可审核、通过或拒绝申请
- **状态跟踪**：实时查看申请处理状态
- **使用记录**：完整的使用历史记录

### 4. 设备维修系统

- **故障报修**：学生可提交设备故障报告
- **维修处理**：教师处理维修申请
- **维修记录**：完整的维修历史跟踪
- **状态管理**：待处理、处理中、已完成等状态

### 5. 满意度评价

- **使用反馈**：学生对实验室使用体验评价
- **评价统计**：教师查看评价数据和统计
- **改进建议**：基于评价的系统优化

### 6. 数据统计与报表

- **实时统计**：可用实验室、使用中实验室、待维修设备等
- **数据导出**：支持 Excel 格式数据导出
- **图表展示**：直观的数据可视化

## 🏗️ 系统架构

### 组件化设计

```
src/
├── components/          # 公共组件
│   ├── header/         # 头部导航组件
│   └── Pagination/     # 分页组件
├── views/              # 页面组件
│   ├── home/           # 首页
│   ├── login/          # 登录页面
│   ├── lab/            # 实验室管理
│   ├── application/    # 申请管理
│   ├── repair/         # 维修管理
│   ├── satisfaction/   # 满意度评价
│   └── apprecord/      # 申请记录
├── stores/             # 状态管理
│   ├── user.js         # 用户状态
│   ├── lab.js          # 实验室状态
│   ├── repair.js       # 维修状态
│   └── application.js  # 申请状态
├── router/             # 路由配置
└── utils/              # 工具函数
```

### 状态管理

- **用户状态 (user.js)**：用户信息、角色权限、登录状态
- **实验室状态 (lab.js)**：实验室列表、设备信息、统计数据
- **维修状态 (repair.js)**：维修申请、处理记录、状态跟踪
- **申请状态 (application.js)**：申请记录、审核流程、使用历史
